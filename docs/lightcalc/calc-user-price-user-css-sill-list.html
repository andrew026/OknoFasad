Подключение калькулятора к html странице

Для подключения калькулятора, поместите в произвольном месте раздела <body> элемент с html кодом:
<div id="lwcdiv"></div> - в этом элементе будет размещен калькулятор.
В разделе <head> нужно инициализировать скрипт калькулятора. В простойшем случае, это можно сделать одной строкой:
<script src="http://www.oknosoft.ru/lightcalc/js/lightcalc.js"></script>
